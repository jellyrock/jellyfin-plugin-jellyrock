Cold launch casting

When enabled, a Roku with JellyRock installed appears in "Play On" even while the app is closed. Selecting it wakes the Roku and starts JellyRock. When disabled, only Rokus with JellyRock already open appear.
When enabled, sideloaded builds (shown as "JellyRock (dev)") also appear as closed cast targets. When disabled, only published installs do, which is useful if you run test or CI devices whose dev builds you never cast to. Casting to a dev build while it is open still works either way.
Changes take up to 30 seconds to take effect.
